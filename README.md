# ProfitPilot — AI Commerce Intelligence

**Don't react to losses. Predict them.**

AI-powered dashboard for Shopify merchants that:
- Calculates TRUE profit after all hidden costs (COGS, shipping, ads, platform fees)
- Predicts stockouts using historical sales + seasonal/holiday patterns
- Forecasts cost increases from real-world events (oil, cotton, logistics)
- Benchmarks pricing against competitors
- AI Copilot powered by Claude for instant store insights

Built at Growzilla × Forest City Hackathon | Powered by Claude
